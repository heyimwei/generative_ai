這是一個使用 Bing Image Creator（基於 Diffusion Model）生成的像素風格圖片集，主題圍繞著可愛的浣熊與小狗角色，探索各種場景與風格。

## 📂 資料夾內容

| 資料夾名稱 |
|------------|
| `夜晚的像素藝術城市` |
| `未來科技火車站` |
| `浣熊小狗在公園裡野餐` |
| `浣熊小狗探索神秘的廢墟` |
| `溫暖舒適的咖啡店` |

每個資料夾中皆包含 4 張由 AI 生成的像素風圖片。

## 🛠 使用工具

- Bing Image Creator  (https://www.bing.com/images/create)
- Prompt 設計以英文撰寫，並加入 pixel art 關鍵詞
